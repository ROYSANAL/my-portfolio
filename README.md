# Sanal Roy - Portfolio

A modern, responsive, and dynamic portfolio website showcasing my experience, projects, and skills as a Backend & Full Stack Developer.

## Features
- **Next.js 15 (App Router)** for fast, optimized rendering.
- **Tailwind CSS v4** for modern, responsive, and maintainable styling.
- **Framer Motion** for smooth scroll animations, hover effects, and a dynamic 3D feel.
- **next-themes** for seamless light/dark mode switching.
- **Responsive Design** adapting flawlessly from mobile screens to 4K displays.
- **Dynamic Backgrounds** with CSS keyframe-based animated floating blobs.

## Setup & Running Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/ROYSANAL/my-portfolio.git
   cd my-portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```
   Open [http://localhost:8000](http://localhost:8000) to view the application in your browser.
